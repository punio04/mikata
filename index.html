    株式会社ハウスメンテ北陸（以下、「当社」といいます。）は、当社が提供するサービスをご利用になるお客さまおよびパートナーさま（以下、「お客さま等」といいます。）の個人情報を適切に取り扱うため、以下のプライバシーポリシーを定めます。当社は、個人情報の保護に努めるとともに、関連する法令およびガイドラインを遵守いたします。

    1. 1.取得する個人情報について

      当社は、以下の方法で個人情報を取得します。

      （1）お客さまの個人情報

      • ・氏名、住所、電話番号、メールアドレスなど、特定の個人を識別できる情報。
      • ・サービスの利用に関連して発生する情報（例：購入した商品やサービス、利用履歴、Cookie、IPアドレス、ブラウザ情報、位置情報など）。
      • ・当社のサービスに投稿された写真、動画、コメント、評価などの情報。

      （2）パートナーさまの個人情報

      • ・氏名、生年月日、性別、職業、住所、電話番号、メールアドレスなど。
      • ・パートナー登録の際の本人確認情報。
      • ・販売した商品やサービス、取引履歴に関連する情報。

    2. 2.個人情報の取得方法について

      当社は、以下の方法でお客さま等の個人情報を取得します。

      1. 1.直接取得：サービス利用時の登録や問い合わせを通じて提供いただく情報。
      2. 2.自動取得：ウェブサイトの利用に伴い収集されるCookieやアクセスログ。
      3. 3.第三者からの取得：提携先や業務委託先を通じて提供される情報。

    3. 3.個人情報の利用目的について

      当社は、以下の目的で個人情報を利用します。

      1. 1.サービス提供および運営のため。
      2. 2.サービスに関するご案内やお問い合わせ対応のため。
      3. 3.利用状況の分析やサービスの改良・改善、新サービスの開発のため。
      4. 4.代金の請求、支払い処理および確認のた
      5. 5.調査協力依頼やイベント案内のため。
      6. 6.トラブル解決や不正利用防止のため。
      7. 7.お客さまに適した広告配信およびカスタマイズのため。

    4. 4. 個人情報の管理について

      当社は、個人情報の漏えい、紛失、改ざん、不正アクセスを防止するため、適切な管理体制を整備します。

      • ・社内規定の制定と従業員教育の実施。
      • ・技術的対策（アクセス制限、暗号化など）の導入。
      • ・委託先を含む管理体制の定期的な見直し。

    5. 5.個人情報の第三者提供について

      当社は、以下の場合を除き、個人情報を第三者に提供いたしません。

      1. 1.お客さまの同意がある場合：商品やサービス提供に必要な場合など。
      2. 2.法令に基づく場合：公的機関からの要請がある場
      3. 3.業務委託先への提供：運営に必要な範囲内で適切な管理を行う委託先への提供。
      4. 4.統計情報としての利用：個人を識別できない形式でのデータ分析や公表。

    6. 6.個人情報の開示・訂正・利用停止・削除

      お客さまがご自身の個人情報の開示、訂正、利用停止、削除を希望される場合は、下記のお問い合わせ窓口までご連絡ください。本人確認後、法令に基づき適切に対応いたします。

    7. 7.プライバシーポリシーの改定について

      当社は、法令の変更や運営方針の見直しに伴い、プライバシーポリシーを改定することがあります。重要な変更がある場合は、当社ウェブサイト上でお知らせいたします。
